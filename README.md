# LTF Launcher - Abandonware France
Launcher en court de développement pour Abandonware France

LFT Launcher contiendra ces fonctionnalités :
- Accès aux téléchargements directement dans le launcher
- Mises a jour automatiques
- Discord Rich Presence
- Accès aux forums avec son compte Abandonware Forums
- Système de traduction et possibilité de changer la langue dans les paramètres
- Bibliothèque inclus pour gérer ses jeux
- Système de succès à gagner pour les jeux disponibles
- Possibilité d'ajouter d'autres jeux se lançant avec un .bat ou un .exe
- Système de succès et de comparaison de temps de jeu si un compte Abandonware France est lié

Le code-source est aussi disponible pour personnaliser votre launcher

LTF Launcher est développé sous Unreal Engine 4.25.4

Rejoignez [Abandonware France](URL 'https://www.abandonware-france.org/') et [Abandonware Forums](URL 'https://www.abandonware-forums.org/')
